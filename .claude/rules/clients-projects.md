# Working with Clients & Projects

When working with files in a client or project folder:

1. **Read the folder's CLAUDE.md first** — it has the full context about who this client is, what you do for them, and active work.

2. **Route content to the right place:**
   - Meeting notes → `notes/`
   - Important email threads → `emails/` (clients only)

3. **Log to memory** — if you create content or make decisions for a client/project, log it to `context/memory/sessions/` with the client/project name so it's searchable.

4. **Match the client's brand** — if the client's CLAUDE.md has brand voice guidelines, follow them when creating content for that client. Their voice, not the user's.
